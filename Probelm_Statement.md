# Problem Statement 1:
A financial company needs a secure file transfer system to exchange confidential reports between branches. To ensure encryption and integrity, the system must use TLS for secure communication.

# Problem Statement 2:
A video streaming platform wants to optimize its DNS resolution process to reduce latency when retrieving video content from different servers. The system must resolve domain names efficiently while implementing caching to minimize redundant queries.

# Problem Statement 3:
You need to implement a reliable file transfer system using the Transmission Control Protocol (TCP). The system should allow a client to upload a file to the server and download a file from the server.

# Problem Statement 4:
You are tasked with developing a simple chat application that allows multiple clients to communicate with a server using the User Datagram Protocol (UDP). The server should broadcast messages received from any client to all connected clients.
