# Network Protocol Implementation & Security Analysis

This repository showcases advanced networking implementations focused on security, concurrency, and performance using Python.

## 🚀 Key Features
- **Security (mTLS):** Implementation of Mutual TLS for end-to-end encrypted file transfers using OpenSSL-generated certificates.
- **DNS Performance & Privacy:** Tools for parallel DNS resolution, DNSSEC validation, and DNS-over-HTTPS (DoH).
- **Concurrent Systems:** Multi-threaded TCP servers handling simultaneous file operations and UDP-based real-time broadcast chat.

## 🛠️ Tech Stack
- **Language:** Python 3.13
- **Libraries:** `socket`, `ssl`, `threading`, `dnspython`, `requests`
- **Security Tools:** OpenSSL
